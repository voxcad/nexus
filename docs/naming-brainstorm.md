# NEXUS Naming Brainstorm

## Context for Name Generation

**What the product IS:**
- Browser-based, all-in-one spatial engineering platform
- Replaces AutoCAD + Civil 3D + Revit + ArcGIS Pro + QGIS in ONE tab
- API-first: every operation is an API call
- Agentic: AI agents are first-class users alongside humans
- Plugin-based: extensible by third parties
- Portable: runs anywhere a browser runs (desktop, tablet, field)
- As-a-Service: can be self-hosted or cloud-hosted
- Local-first: works offline, syncs when connected

**Core attributes the name must convey:**
1. Web-based / modern / cloud-native
2. All-in-one / unified / convergence
3. Agentic / AI-native / intelligent
4. Memorable / short / easy to type and say
5. API-based / developer-friendly / programmable
6. Plugin ecosystem / extensible / platform
7. Portable / runs anywhere
8. Professional / engineering-grade / trustworthy

**Name constraints:**
- 1-2 syllables ideal, 3 max
- No hyphens in org name (GitHub allows but looks dated)
- Available as GitHub org
- .dev or .io domain would be nice
- Should sound good in: "I built this in ___", "Deploy your ___ agent", "___ API"

**Inspiration from successful dev tool names:**
- Vercel (invented word, sounds like "verse" + "cell")
- Supabase (super + base)
- Prisma (light/clarity)
- Figma (figure + schema)
- Miro (mirror/reflection)
- Stripe (clean, one word)
- Linear (direct, efficient)
- Render (what it does)
- Railway (infrastructure)
- Turso (short, invented)

## Top 5 Recommendations

1. **Topos** — Short, memorable, Greek for "place." Works in every context: "Topos API", "Deploy a Topos agent", "Built with Topos." Sounds modern and clean.

2. **Strato** — Layers are fundamental to everything we do (CAD layers, GIS layers, BIM layers). Short, strong, memorable. "Strato Platform", "Strato CLI."

3. **Locus** — Latin for "place." Professional, engineering-grade sound. "Locus API", "Locus agents." 4 letters.

4. **Karta** — Map in Swedish/Russian/many languages. Universal. "Karta Platform", "Karta agents." Unique in tech.

5. **Datum** — Every engineer knows what a datum is. Reference point. Foundation. "Datum Platform", "Datum API." Might conflict with database "data" associations though.
